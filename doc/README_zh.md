## 🚩 我们的使命

在当今高度中心化的数字世界中，我们的数据被商品化，我们的声音被科技巨头垄断和审查。我们坚信，言论自由、数据隐私和数字权利是基本人权。

**Magpie** 是一个为抵抗审查和打破数据孤岛而构建的去中心化协议和社交网络。我们正在创造一个数字公共广场，在这里，您控制您的身份，您拥有您的数据，您的言论是自由的。

## 🌟 核心价值观

首先，这是一个由价值观驱动的项目。我们所有的技术决策都基于以下原则：

* **自由 (Freedom):** 我们捍卫言论自由和信息的自由流动。本网络的设计核心是抗审查，没有任何中心化权威可以单方面删除内容或封锁用户。
* **隐私 (Privacy):** 隐私是不可侵犯的权利。我们严格遵守高级别隐私标准，确保用户拥有对其数据的完全控制权。
* **去中心化 (Decentralization):** 网络的未来应由其用户共同决定，而非由公司董事会决定。自托管和 P2P 技术确保没有任何单一实体可以控制网络。
* **平等 (Equality):** 这是一个向所有人开放的、包容的网络。我们尊重所有人的身份和表达。

## ✨ 核心功能

* **🛡️ 安全通信:**
    * **端到端加密 (E2EE):** 1:1 聊天和群聊采用 Signal 协议。
    * **消息送达保证:** 确保消息可靠送达。
    * **离线存储:** 消息存储在本地，并在上线时同步。
    * **状态显示:** 在线/已读/正在输入等状态。

* **🔐 数据主权与隐私:**
    * **自托管:** 您可以运行自己的节点。如果不是本地部署，也可以选择托管服务商，除非相互连接，否则支持去中心化。
    * **去中心化支持:** 备份消息历史记录并将其永久存储在您选择的任何地方，允许随时恢复。
    * **标签与排序:** 轻松管理您的对话。

* **🌐 抗审查:**
    * **P2P 网络:** 节点之间直接通信，中心化实体无法关闭网络。
    * **客户端过滤:** 您自己控制您想看的内容。我们提供强大的客户端工具（非平台级审查）来屏蔽、过滤您不想看到的内容或用户。

* **💬 全功能社交体验:**
    * **1:1 聊天与群聊:** 全功能的即时通讯。
    * **二手市场:** 内置市场功能。
    * **附加组件:** 实施严格的安全措施和加密以防止数据泄露。

## 🌍 跨平台支持

我们的目标是让 Magpie 随处可用。

* **💻 桌面端:** Windows, macOS, Linux (Electron / .NET)
* **📱 移动端:** Android, iOS (Flutter, Kotlin, Swift)
* **🌐 网页端:** 所有现代浏览器 (Vue, Node.js, Nuxt, TypeScript) - 无需安装。

## 🛠️ 架构与技术栈

Magpie 的核心是一个去中心化的 P2P 协议。为了确保高性能、可扩展性和丰富的开发者生态，任何人都可以使用以下技术来运行高性能的节点或辅助服务。

* **Web:** Web 客户端 + 管理后台 (Vue, Node.js, Nuxt, TypeScript)
* **小程序:** UniApp
* **Android:** Kotlin & C++
* **iOS:** Swift
* **跨平台移动端:** Flutter
* **桌面端:** Electron / .NET
* **后端服务:** Java + Python + Go + C++
